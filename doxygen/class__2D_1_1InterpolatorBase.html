Set references to the interpolated data to memory outside the interpolator. This is potentially unsafe. The interpolator will not take ownership of the memory, the caller will still be required to free it when it is no longer needed. Freeing memory before calling the interpolator may cause uninitialized memory access and is undefined behavior.

Modifying the data after setting the interpolator references is undefined behavior as some interpolators do some setup work that will be negated when the data has changed.
