Linearize vehicle model using current speed.

Linear bicycle model with extended 2nd order steering dynamics

dx/dt = A*x + B*u

x = [y, vy, yaw, yawrate, delta, d(delta)/dt] u = [0, 0, 0, 0, 0, input_delta]
